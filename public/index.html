FAQ rapide

Mes données sont-elles sauvegardées ?

Oui, les profils validés sont stockés dans la collection users de Firestore.

Que faire si je ne reçois pas le code OTP ?

Vérifie le format international du numéro puis relance l’envoi depuis la page de connexion/inscription.

Puis-je compléter mon profil plus tard ?

Oui, après connexion tu es redirigé vers la page profil si ton compte n’est pas encore complété.
